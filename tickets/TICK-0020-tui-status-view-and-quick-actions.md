# Title

Build a TUI status view with workflow quick actions

## Status

todo

## Priority

medium

## Type

feature

## Summary

Add a terminal UI layer that makes task state, latest artifacts, and next-step actions easier to scan and operate.

## Scope

- Show current role, round, and status in a TUI view
- Surface recent artifacts like `review.md` and `developer-note.md`
- Add quick actions such as continue, inspect review, approve and stop, or open blocking issues only

## Out of Scope

- Full graphical desktop UI
- Web dashboard

## Acceptance Criteria

- Users can inspect task progress through a TUI-oriented view
- Common next actions are easier than typing raw commands repeatedly
- The TUI reflects the same underlying task state as CLI commands
