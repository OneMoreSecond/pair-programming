## Status

## Summary

Extend the TUI so users can scan recent artifacts and round history without leaving the terminal status experience.

## Scope

- Surface recent artifacts such as `review.md` and `developer-note.md`
- Add round-history browsing within the TUI
- Reuse existing artifact and history data sources

## Out of Scope

- Executing workflow actions from the TUI
- Full-screen editors or file opening integration

## Acceptance Criteria

- Users can inspect recent artifacts from the TUI
- Users can browse round history from the same TUI surface
- The data shown matches existing CLI artifact and history commands
